# 🌍 Afri Educ Tech - Immersive Learning Platform

![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![React](https://img.shields.io/badge/React-2026-61DAFB?logo=react)

**Afri Educ Tech** is a cutting-edge EdTech platform designed to redefine the educational landscape in Africa. By integrating immersive 3D technologies and interactive virtual laboratories, we empower students to master complex concepts through visualization and manipulation.

---

## ✨ Key Features

* **Immersive 3D Viewer**: Interactive zone for manipulating anatomical, technical, and scientific models.
* **High-Fidelity UI**: A modern, responsive interface built with a mobile-first approach.
* **Performance Optimized**: Designed to run smoothly even in areas with limited bandwidth.
* **Universal Navigation**: A unique, accessible menu system designed for both Desktop and Mobile users.

## 🎨 Visual Identity

The project follows a specific color palette to reflect innovation and trust:
* Primary Blue: `#73C2F0` (Vivid Azure)
* Background: `#EAF6FA` (Soft Sky)
* Action Orange: `#BA3A0F` (Burned Orange)
* Typography: **Outfit** (Modern Sans-Serif)

---

## 🛠️ Tech Stack

* **Frontend**: [React.js](https://react.dev/)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Utility-first framework)
* **Routing**: [React Router Dom](https://reactrouter.com/)
* **Build Tool**: [Vite.js](https://vitejs.dev/)

## 🚀 Getting Started

### 1. Clone the repository
```bash

### 2. Install dependencies
```bash

### 3. Configure Tailwind
```bash
@import "tailwindcss";

### 4. Run the development server
